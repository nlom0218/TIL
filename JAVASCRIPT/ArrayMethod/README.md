# Array Method

배열에는 다양한 메서드가 존재한다. 메서드 중에서는 고차 함수도 다수 존재한다. 또한 어떤 메서드는 기존 배열을 바꾸기도 하고 어떤 메서드는 기존 배열을 바꾸지 않으면서 새로운 배열을 반환하기도 한다.
다양하게 나눌 수 있는 배열의 메서드지만 11개의 범주로 나누어 정리한다.
